# changedetection.io

> Website change monitoring with a browserless companion service

## Stack Role

This stack directory stores the `compose.yaml`, `README.md`, and tracked `.env.example` for `changedetection-io`. For the encrypted deployment workflow with SOPS, age, File Watcher, and Komodo, see [`docs/sops-age-komodo.md`](../docs/sops-age-komodo.md).

## Services

- `browserless-v2`
- `changedetection-io`
